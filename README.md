Adshub.io
Adshub.io is an innovative advertising platform that simplifies ad creation, distribution, and performance tracking for businesses, creators, and marketers. 
It acts as a central hub where users can launch targeted campaigns, monitor real-time results, and optimize reach without complex setups—empowering anyone to 
scale their advertising efforts effortlessly.

🚀 Features
1. Campaign Builder: Intuitive drag-and-drop interface for creating targeted ad campaigns with A/B testing support.
2. Real-Time Analytics: Live dashboards tracking impressions, clicks, conversions, and revenue.
3. Ad Network Integration: Connects with multiple platforms for unified management.
4. Smart Targeting: Audience segmentation based on demographics, behavior, and geolocation.
5. Monetization Tools: Revenue sharing models and automated payouts.
6. Responsive Design: Optimized for web, mobile, and embedded widgets.
7. API-First: RESTful APIs for custom integrations and webhooks for events.

•How It Works
Users start by signing up and creating campaigns through an intuitive dashboard—selecting ad formats, audiences, and budgets via simple drag-and-drop tools. 
The platform then automatically distributes ads across networks, matching them to relevant users based on behavior and location for maximum impact. Real-time 
analytics show key metrics like clicks, views, and conversions, with smart suggestions to refine future campaigns. Payouts and revenue sharing happen seamlessly,
making it ideal for scaling ad operations effortlessly.

•Technologies used
1. Node/Express for backend
2. React and tailwind-css for frontend
3. NeonDB for serverless postgres SQL
4. Prisma ORM for database queries
5. clerk for authentication
6. sentry for error tracking
7. Gemini API for image and video generation
